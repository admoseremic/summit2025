A state change derived field helps you measure how often users move from “silver” to “gold” and see which marketing channels most contribute to upgrades. You can create a segment of users who have upgraded.

While calculated metrics can't be used in an experiment, segment, funnel, etc. A "derived" metric can be! Create one using the Math function in derived fields.
